# 🧮 Calculadora em C#

## 📌 Descrição
Este projeto consiste em uma **calculadora desenvolvida em C#** que permite ao usuário realizar operações matemáticas básicas através do terminal.

O programa apresenta um menu interativo onde o usuário pode escolher a operação desejada e inserir os números para realizar o cálculo.

As operações disponíveis são:
- Adição
- Subtração
- Multiplicação
- Divisão

O sistema também possui **tratamento de erro para divisão por zero**, garantindo que o programa não execute uma operação inválida.

---

## ⚙️ Funcionalidades

- Menu interativo no console
- Operações matemáticas básicas
- Entrada de números pelo usuário
- Tratamento de erro para divisão por zero
- Opção de sair do programa

## ▶️ Como Executar o Projeto

1. Clone o repositório:
```
git clone https://github.com/seu-repositorio/calculadora.git
```

2. Abra o projeto no **Visual Studio**.

3. Execute o projeto pressionando:

```
F5
```

4. Escolha a operação desejada no menu e digite os números para realizar o cálculo.
